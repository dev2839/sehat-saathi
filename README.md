# 🩺 Sehat Saathi – Child Health Record System

## 📌 Overview
Sehat Saathi is a comprehensive child health management system designed to assist field health workers in collecting, managing, and analyzing critical health data, especially in remote and low-connectivity environments.

Built as part of the MOSIP Decode Challenge 2025, the application focuses on reliability, accessibility, and real-world usability by enabling offline data collection with seamless synchronization when connectivity is restored.

---

## 🚀 Key Highlights

- 📡 Offline-first system for remote data collection  
- 🔐 Secure authentication with MOSIP eSignet integration  
- 📍 GPS-based location tracking for accurate records  
- 👶 Unique health ID generation for each child  
- 📸 Photo capture and health data storage  
- 📊 Automated BMI calculation and categorization  
- 🔄 Real-time data synchronization when online  
- 💻 Admin dashboard for monitoring and analytics  

---

## 🎯 Purpose

This project is designed to solve real-world challenges faced by healthcare workers in underserved regions. It ensures that critical child health data can be collected reliably without depending on constant internet access, improving efficiency and data availability for better healthcare decisions.

---

## 🏗️ System Overview

The platform consists of:
- A field-side interface for data collection  
- An administrative dashboard for analysis and monitoring  
- A backend system for secure data storage and synchronization  

---

## 📊 Features

- Role-based access (Field Workers & Administrators)  
- Offline data storage with auto-sync  
- Health record management  
- Real-time feedback and notifications  
- Scalable data handling for large datasets
